FYI: This works better only in linux as gunicorn is a part of it.

To Run:
## git clone github.com/ajaysiva07/django-gunicorn-nginx-docker-compose-starter.git
## cd django-gunicorn-nginx-docker-compose-starter
## docker-compose up -d
